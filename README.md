# Уютные блоки (Cozy Blocks)

Казуальная головоломка с блоками для браузера. [GitHub](https://github.com/lleeeuuuuanchik/cozy-blocks) Размещай фигуры на сетке 10×10, собирай линии, проходи уровни и зарабатывай монеты. Публикуется в [Яндекс Играх](https://yandex.ru/games/).

## Запуск локально

Без сборки — открой `index.html` в браузере или запусти локальный сервер:

```bash
python3 -m http.server 8000
# Открой http://localhost:8000
```

## Как залить игру в Яндекс Игры

### 1. Собери ZIP-архив

В корне архива должны быть: `index.html`, `styles.css`, папки `src/` и `assets/`.

```bash
zip -r uyutnye-bloki.zip index.html styles.css src/ assets/ -x "*.DS_Store" -x ".git/*" -x "*.md" -x ".cursor/*" -x ".claude/*"
```

Проверь: распакуй архив и открой `index.html` — игра должна запускаться.

### 2. Загрузи в консоль

1. Зайди в [Консоль Яндекс Игр](https://yandex.ru/dev/games/console/)
2. Создай новую игру или открой черновик
3. Загрузи `uyutnye-bloki.zip` (дождись проверки файла, до ~15 мин)
4. Заполни: название, описание, SEO, «Об игре», «Как играть», иконка (512×512), обложка (800×470)
5. Сохрани и отправь на модерацию

### 3. Сроки

Модерация обычно занимает **3–5 рабочих дней**. После отклонения — минимум 24 часа до повторной отправки.
