# Промпт для новых игр под Яндекс Игры

Скопируй этот текст в начало чата при создании новой игры:

---

**Делаю игру для Яндекс Игр.** Учитывай при разработке:

1. **SDK** — подключать только на домене Яндекс (yandex.ru, yandex.com и т.п.). Локально не загружать, иначе ошибки "No parent to post message".
2. **LoadingAPI.ready()** — вызывать после полной загрузки (меню/геймплей готов). Обязательно для модерации.
3. **Реклама** — rewarded и interstitial. Без SDK игра должна работать без рекламы.
4. **ZIP для выгрузки** — в корне: index.html, styles.css, src/, assets/. Не вкладывать в подпапку.
5. **README** — инструкция по запуску и загрузке в консоль Яндекс Игр.
6. **Модерация** — 3–5 рабочих дней. ZIP проверяется до ~15 мин; отправлять на модерацию только после проверки.
